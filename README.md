# API REST Didática — Ambiente de Aulas

## Estrutura
```
Explicacoes/
├── backend/     → Node.js + Express (porta 3001)
└── frontend/    → Next.js 14 (porta 3000)
```

## Iniciar

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
